# Retail Analytics with Pandas

Data analysis project processing 2.1M+ retail transactions for customer and product insights.

## Overview

Analysis of retail transaction data demonstrating large-scale data processing, memory optimization, and business intelligence extraction. Processes 2.1M+ transactions covering 2,099 households and 84,138 products with $6.6M in total sales.

## Dataset

- **2,157,433 transactions** across multi-week period
- **2,099 households** with purchase history
- **84,138 products** (bikes, helmets, accessories)
- **$6.6M total sales** with discount tracking

## Key Features

- Memory optimization (25% reduction through type casting)
- Customer lifetime value analysis
- Product performance ranking
- Discount effectiveness analysis
- Customer segmentation

## Technology Stack

| Component | Technology |
|-----------|------------|
| **Language** | Python 3.x |
| **Processing** | Pandas, NumPy |
| **Visualization** | Matplotlib |
| **Environment** | Jupyter Notebook |

## Results

**Performance:**
- 25% memory reduction via intelligent type casting
- Zero missing values across all columns

**Business Insights:**
- Average basket value: $28.62
- Average customer lifetime value: $3,175.91
- Top product: Mountain-200 Black 38 ($1.3M revenue)
- Identified high-value customer segments
